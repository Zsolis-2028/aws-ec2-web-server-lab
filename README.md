# Deploying and Configuring a public AWS web server to demonstrate cloud fundamentals for IT roles

## Overview
This project demonstrates how to deploy a public web server on AWS using EC2 and Amazon Linux.

## Technologies Used
- AWS EC2
- Amazon Linux 2023
- Apache (httpd)
- Security Groups

## What I Did
- Launched an EC2 instance using Free Tier
- Connected using EC2 Instance Connect
- Installed and started Apache web server
- Enabled the service to start on boot
- Verified public access using the instance public IP

## What I Learned
- Differences between Linux distributions
- How EC2 and security groups work together
- Basic Linux service management
